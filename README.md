# CoffeeShop-DB-Manager-v2.2-
A robust, command-line-based Coffee Shop Management System built with Python and MySQL. This version (2.2) focuses on backend integration, allowing for persistent data storage of orders, inventory, and billing records.

Database Persistence: Unlike basic scripts, this project uses MySQL to ensure that inventory data and customer orders remain saved after the program closes.

Dynamic Billing: Automated calculation of order totals, GST, and service fees directly from database-stored prices.
